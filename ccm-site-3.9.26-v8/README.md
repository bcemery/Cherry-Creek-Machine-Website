# Cherry Creek Machine — Website

ISO 9001:2015 certified CNC machine shop in Tulsa, Oklahoma.  
Website: [ccmachine.com](https://www.ccmachine.com)

## Site Structure

```
index.html          → Homepage
cnc-milling/        → CNC Milling services
cnc-turning/        → CNC Turning services
waterjet-cutting/   → Waterjet Cutting services
quality/            → Quality & ISO certification
equipment/          → Equipment list
about/              → About the team
careers/            → Careers & job applications
thank-you/          → Form submission confirmation
capabilities/       → Redirect to CNC Milling
```

## How to Update the Site

1. Edit any HTML file in this repository
2. Commit and push your changes to the `main` branch
3. Netlify automatically rebuilds and deploys within ~30 seconds

## Forms

All forms use **Netlify Forms** (detected automatically during build).  
Submissions appear in: Netlify Dashboard → Forms.

## Hosting

- **Host:** Netlify (auto-deploys from this repo)
- **DNS:** Configured through your domain registrar
